## Install

```bash
# use npm
$ npm install @gitmars/go --save

# use yarn
$ yarn add @gitmars/go

# use pnpm
$ pnpm install @gitmars/go
```

## Usage

```js
const { getUserInfo } = require('@gitmars/go')

getUserInfo()
```
