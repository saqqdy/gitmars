## Install

```bash
# use npm
$ npm install @gitmars/hook --save

# use yarn
$ yarn add @gitmars/hook

# use pnpm
$ pnpm install @gitmars/hook
```

## Usage

```js
const { getUserInfo } = require('@gitmars/hook')

getUserInfo()
```
