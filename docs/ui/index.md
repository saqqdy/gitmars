## gitm ui <Badge text="beta" type="warning"/>

如果你厌倦了敲指令，gitmars提供了懒人UI界面

-   使用：`gitm ui [-p --port [port]]`
-   传值：

| 名称   | 简写 | 说明       | 类型   | 可选值 | 传值必填 | 默认 |
| ------ | ---- | ---------- | ------ | ------ | -------- | ---- |
| --port | -p   | 启动端口号 | Number | -      | 否       | 3000 |

-   示例：

```shell
gitm ui --port 3000
```
